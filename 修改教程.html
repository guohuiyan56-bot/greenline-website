GlobalTrade 修改教程

90% 的日常改动只需要编辑 data.js 一个文件

打开文件 → 改文字 → 刷新浏览器 → 搞定

快速上手

核心原则：这个网页是纯静态 HTML，所有内容由 data.js 动态注入。所以你改 data.js 里的任何一个中/英文字段，刷新页面就能看到效果，不需要编译、不需要跑构建。

文件职责一览

文件改的频率里面有什么
data.js⭐⭐⭐⭐⭐ 最常改产品、分类、公司信息、联系方式、合作伙伴、客户评价、流程步骤、统计数据
i18n.js⭐⭐ 偶尔改按钮文字、标题文字、提示文案（少量 UI 文案）
index.html CSS 部分⭐ 少改配色、字号、间距、圆角（在 :root 变量里）
index.html HTML 部分极少改页面结构、区块顺序
app.js一般不改渲染引擎、动画系统、视差滚动、语言切换逻辑

怎么预览

启动本地服务器终端
# 在 GlobalTrade-site 目录下运行
python3 -m http.server 8000
# 浏览器打开 http://localhost:8000

或者用 VS Code 的 Live Server 插件、Nginx、任何静态文件服务器都行。无论如何不要直接用 file:// 协议打开，部分功能在 file 协议下会失效。

data.js 是一个 JavaScript 对象 GT_DATA = { ... }。里面每个字段几乎都是双语的（_cn 和 _en 后缀）。

改配色 — 全局调色板

打开 index.html，找到 <style> 标签最顶部的 :root。改了这里的变量，全站所有地方的颜色都会跟着变。

index.html → :root 变量改颜色只改这里
--c-primary: #C4956A;        /* 焦糖主色 → 按钮、链接、高亮 */
--c-primary-h: #D4A574;      /* 浅焦糖 → hover 状态 */
--c-primary-d: #8B5E3C;      /* 深咖色 → 滚条、底部强调 */
--c-accent: #6B3A2A;         /* 浓缩咖啡 → 进度条渐变终点 */
--c-bg: #0F0A07;            /* 页面底色（最深的咖啡底） */
--c-bg-alt: #15100C;         /* 交替区块底色 */
--c-bg-card: #1C1510;        /* 卡片底色 */
--c-bg-surface: #221913;     /* 输入框底色 */
--c-text: #FFF8F0;           /* 主文字色（暖白） */
--c-text-dim: #B8A898;       /* 次要文字色 */
--c-text-muted: #7A6E62;     /* 弱化文字色 */
快速配色方案：如果你想要金色调，把 --c-primary 改成 #C8A45C；想要蓝灰调就改成 #7B9CB5。只需要改这一个变量，其他辅色可以保持不动，整体风格就全变了。

此外还有：

变量含义默认值
--font-body正文/UI 字体'Inter', 'PingFang SC', sans-serif
--font-display标题/展示字体（衬线）'Playfair Display', serif
--radius-sm/md/lg/xl圆角大小6 / 12 / 20 / 28 px
--header-h导航栏高度78px

修改/新增产品

data.js → products
products: [
  {
    id: "p01",                          ← 唯一 ID
    category: "machinery",             ← 必须和上面 categories 的某个 id 一致
    name_cn: "智能数控机床",           ← 产品中文名
    name_en: "Smart CNC Machine",
    desc_cn: "高精度五轴联动...",    ← 简短描述（两行以内）
    desc_en: "5-axis high-precision...",
    moq: "1 台",                     ← 起订量
    image: ""                          ← 图片路径（见"加图片"章节）
  },
  // 再加一个产品：复制上面整个 { ... } 块，改 id 和字段值，粘贴在数组里就行
]
新增产品三步走：① 复制一个已有产品块；② 改 id（别跟已有的重复）；③ 改 category、name、desc。分类标签和筛选是自动的，不用额外配置。

加图片

产品图片

三步加产品图
第 1 步：把图片放进 assets/img/
assets/img/cnc-machine.jpg

第 2 步：在 data.js 找到对应产品，填 image 字段
{ id:"p01", ..., image: "assets/img/cnc-machine.jpg" }

第 3 步：刷新浏览器
图片建议：产品图用 4:3 或 1:1 比例，宽度 600-800px 就够了（网页会缩放）。格式用 JPG 或 WebP。如果 image 字段为空，页面会自动显示该分类的 emoji 占位图标。

目前图片占位是用 emoji + 渐变背景。如果你想加 Hero 背景图或"关于我们"图片，需要改 index.html 对应区块的 CSS（加 background-image）。这部分改动稍复杂，遇到具体需求可以来问我。

调动画效果

控制滚动入场动画速度

所有入场动画由 CSS 控制，在 index.html 的 <style> 标签内：

index.html → 动画 CSS
/* 基础入场动画（改 transition 的时长就可以快/慢） */
.reveal-up {
  transform: translateY(60px);        ← 起始偏移量，改大 = 弹得更远
  transition: all 0.85s var(--ease);  ← 0.85s = 持续时间，改大 = 更慢
}

/* 交错动画每层的延迟 */
.delay-1 { transition-delay: 0.08s }  ← 第 1 个的延迟
.delay-2 { transition-delay: 0.16s }
.delay-3 { transition-delay: 0.24s }
// ...最多到 .delay-6

关闭某个动画效果

如果你觉得鼠标跟随光标太花哨，在 app.js 里找到这段注释掉：

app.js → 关闭自定义光标
// 找到这一段（约第 51-60 行），全部注释掉即可
/*
document.querySelectorAll('a, button, ...').forEach(function(el){
  el.addEventListener('mouseenter', ...);

进阶技巧

调整区块顺序

在 index.html 中每个 <section> 就是一个区块。直接剪切粘贴整段 <section>...</section> 就能调整顺序。导航锚点会自动跟上。

中英文切换默认语言

在 app.js 顶部 var lang = 'cn' 改成 var lang = 'en' 就是英文默认。

SEO

这是纯前端渲染，搜索引擎爬虫可能抓不到 JS 动态生成的内容。如果要 SEO，建议加上 <meta> 标签和结构化数据。或者在服务器端预渲染。
