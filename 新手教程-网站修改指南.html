cd /c/Users/李燕/WorkBuddy/2026-07-22-10-24-57/GlobalTrade-site

按回车 → 这行是进入你的网站文件夹

git add -A && git commit -m "update"

按回车 → 这行是保存所有改动

git push

按回车 → 这行是上传到网站

在浏览器地址栏输入：www.greenlineint.com 然后回车。

❓ 常见问题

Q: 我只想改中文，不改英文可以吗？

可以。每个字段都有 _cn（中文）和 _en（英文），你只改中文那边的字就行，英文那边不动。

Q: 我想加新产品，怎么加？

在 products 列表里复制一整行，改好 id（不能重复）、category（要对应上面分类的 id），然后改名字和描述。

Q: 我想加新分类，怎么加？

在 categories 列表里复制一整行，改好 id（不能重复），然后改 name_cn、name_en、icon。

Q: 我想删掉某个产品，怎么删？

在 products 列表里，把那一整行删掉就行。注意删完之后上一行的末尾要有逗号。

Q: 我想改成单语言（不要英文），怎么弄？

不用改代码，网站访问时默认显示中文。右上角的 EN 按钮只是给老外看的，你不管它就行。

Q: push 的时候卡住了怎么办？

按 Ctrl+C 取消，然后重新跑一遍三行命令。如果还是卡住，告诉我。
